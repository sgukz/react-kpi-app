# React Theme REH

Installation

```
$ git clone https://gitlab.com/lek.pramual/react-theme-reh.git
$ cd react-theme-reh
$ npm install # or yarn install
$ npm start
```
Run is React Port.

```sh
http://localhost:3000/
```
